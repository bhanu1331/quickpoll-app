# QuickPoll App

A simple voting application built using HTML, CSS, JavaScript and Node.js.

## Features
- Create polls with 2–4 options
- Vote on polls
- Live results with percentages
- Prevent duplicate voting using localStorage
- Delete polls
- Display total votes and winner

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)

## How to Run

### Backend
cd backend  
node server.js  

### Frontend
Open index.html in browser
